🙋 What is this?

Generate HTML code optimal for social media, SEO, mobile for your single-page app. This uses web standards from Open Graph (Facebook) and Twitter to provide optimal results. Also generates social media posts.

Features:

  • Generates standard meta tags based on Open Graph (Facebook), Twitter, and SEO best practices.
  • This stores your last generated form (in localStorage) so when you get back to it, it remembers what you entered last.
  • Yes this is 👨‍💻 Open Source!

🎥 Demo video & some thoughts

🤷 Why?

I haven't found a good enough tool that would do this basic thing as generate text for my meta tags.

To speed up the process of building small apps, I built this. I also saw that other tools available are unnecessarily complicated. If you also build small single-page apps, you'll like this too.
